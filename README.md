# Check this out (이미지 기반 책 정보검색 웹 서비스) 
----------------
Check this out은 이미지 기반 책 정보 검색 웹 서비스로서 사용자가 책 표지의 이미지를 넣었을때 해당 책 표지를 분석하여 책에 대한 카테고리, 제목, 글쓴이, 출판사, 가격을 알려주는 웹 서비스입니다.
